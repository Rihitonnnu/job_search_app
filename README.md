# 就活アプリPROGRAB(プログラブ)

就活生側(ユーザー)と企業側でのマルチログインアプリとなっています。

# 概要

PROGRABには求人に対する応募機能の他に、ユーザーが気になった求人情報を自身のgoogleスプレッドシートにボタンひとつで記録できる機能があります。PROGRABを使えば時間のない就活生でも楽に早く情報を集めることができます！
